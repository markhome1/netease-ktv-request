# music-scanplayer

一个在局域网内运行的扫码点歌服务，当前主路线是“提交网易云歌曲链接，自动写入共享歌单，并让网页版网易云继续顺序播放”。

## 当前能力

- 主机生成二维码，手机扫码即可打开点歌页
- 手机端只提交网易云歌曲链接
- 服务端自动通过请求，并加入本地共享播放队列
- Playwright 控制独立浏览器，把歌曲自动加入指定网易云共享歌单
- 加歌成功后自动回到歌单页并点击播放按钮
- 本地 JSON 持久化当前播放、队列和同步状态

## 当前页面

- 主机管理页：只保留 Playwright 浏览器自动化配置和控制按钮
- 手机点歌页：只保留一个网易云歌曲链接输入框

## Playwright 自动化用法

1. 执行 npm install。
2. 首次使用时执行 npx playwright install chromium。
3. 启动服务后打开 /admin。
4. 在管理页里确认目标歌单名称、浏览器目录等配置。
5. 点击“启动浏览器”，在受控浏览器里手动登录网易云一次。
6. 点击“打开共享歌单”，确认当前落在目标歌单页面。
7. 保持“启用自动写共享歌单”和“加歌成功后自动点击歌单播放”为开启状态。
8. 之后手机端访问 /request，只提交网易云歌曲链接即可。

## 运行

```bash
npm install
npm start
```

启动后访问：

- 主机管理页：/admin
- 手机点歌页：/request

## 说明

- 当前目标歌单和 Playwright 配置保存在 data/state.json。
- Playwright 方案本质上是网页自动化，网易云网页版改版后可能需要调整选择器。
- 官方 OpenAPI 相关文档仍保留在 docs 目录，但当前默认不依赖官方开发平台账号。
